Groupie Tracker


Objectives:

  • A website where we use the given API to display it in the pages of the website and that done using golang to build the web appilaction.
  • The whole project refers to recover data of the artists, like for example their name, the place where they were born and much much more.
  • Our goal was to use this information to help the user find what he wanted to know about some of his favorite artists/groups!

Features:

  • Searching bar: enables searching by the name of the artist.
  • filtering: enables filtering the results to only the artist that matches the search.
  • Geolocalization: enables mapping the different concerts locations of a certain artist/band given by the Client.

Instructions:

  • The backend written in Go.
  • The project can deal have error handler to deal with error.
  • The code written if following the good practice.
  • The project contain test files.
